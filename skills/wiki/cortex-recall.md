# cortex-recall

Responder una pregunta usando el contenido del wiki como fuente.

1. Leer `wiki/index.md` para identificar páginas relevantes
2. Leer las páginas relevantes
3. Sintetizar respuesta con citas a páginas específicas
4. Si falta info, sugerir fuentes para ingerir

## Reglas

- Citar páginas del wiki, no conocimiento de entrenamiento
- Si hay contradicciones entre páginas, señalarlo
- Si el tema no está en el wiki, decirlo explícitamente
